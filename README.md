# EquiTorch: A Modularized Package for Flexibly Constructing Equivariant GNNs
EquiTorch is a PyTorch-based package that provides a modularized and standardized set of operations for constructing equivariant graph neural networks (GNNs). It builds upon the popular Pytorch-Geometric library, aligning with its "Message Passing Neural Network" (MPNN) frameworks.


> ***This is an early-stage release of the EquiTorch package. As such, it may contain untested, undocumented, or unoptimized functions. We apologize for any inconvenience this may cause. We are actively working to improve and expand the package, and will be providing more comprehensive examples and tutorials.***

## Key Features
- Modularized Equivariant Operations: We implements a variety of equivariant operations, including tensor products, linear transformations, activations, and more, allowing researchers to flexibly compose these modules to explore novel equivariant GNN architectures.
- Standardized Data Layout: We establishes a consistent data layout convention for equivariant features and transformations, simplifying the integration of different equivariant techniques.
- Efficient implementation: We tried to optimize some of the implementations of common used operation in equivariant GNNs, leading to improved performance compared to the baseline implementations.
